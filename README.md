# 🛒 Shopping Website (Pure PHP)

## Giới thiệu

Đây là website bán hàng đơn giản được xây dựng bằng **PHP thuần + MySQL** để phục vụ học tập và làm đồ án.

## Chức năng

* Trang chủ hiển thị sản phẩm
* Xem chi tiết sản phẩm
* Trang quản trị (Admin)
* Thêm / Sửa / Xoá sản phẩm

## Công nghệ sử dụng

* PHP
* MySQL
* HTML, CSS
* XAMPP
